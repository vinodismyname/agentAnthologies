# Obsidian Integration Instructions for Agent Anthologies

Please implement the following changes to make this vault work well in Obsidian. Do not modify the actual content of the files, only add metadata and navigation elements.

- Leave `executable-anthology/run_anthology.sh` as is (Obsidian will ignore it)
- Create `executable-anthology/run_anthology_script.md` with the contents of the .sh file wrapped in a bash code block
- Add appropriate frontmatter to the new .md file

## Task 2: Add YAML Frontmatter to All Files

### For all files in `fragments-archive/story-*` folders:
Add this frontmatter pattern (adjust values accordingly):
```yaml

story-number: 1  # Change based on story (1-4)
story-theme: authenticity  # Change to: artifacts, awareness, or identity

---
```
